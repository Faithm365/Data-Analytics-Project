# Data-Analytics-Project
Analysing data for the new CEO to understand the current position of the company based on the historical data of the past 6 months.
# Problem Statement
The Coffee Shop has not been performing well in the past year. This resulted in the change in the management structure and the new CEO was brought in. The new CEO needs to understand the current standing of the company.
# Objectives
The following steps will be followed to reach the aim of this project:
- To perform data inspection
- To identify if the transaction is collected in full.
# Tools used to implement this project
- Snowflake for data processing in SQL.
- Microsoft Excell for creating charts and Graphs
- #Summary of the results
- The analysis shows that more than 60% of the revenue comes from Coffee and Tea sales in the months of May and June.
